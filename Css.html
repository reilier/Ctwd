Introduction to CSS

    
   

What is CSS?

       

CSS (Cascading Style Sheets) is a stylesheet language used to describe the presentation of a webpage. It controls the layout, colors, fonts, and overall visual appearance.

 
 
   

CSS Selectors

       

Selectors target HTML elements to apply styles. Common selectors include element selectors, class selectors, ID selectors, and attribute selectors.

 
 
   

CSS Properties

       

CSS properties define the visual effects applied to elements, such as color, font-size, margin, padding, background-color, and more.

 
 
   

CSS Box Model

       

The CSS box model describes the rectangular boxes generated for elements, consisting of margins, borders, padding, and the actual content area. Understanding it is key to controlling layout and spacing.
